=======================
Working with JupyterLab
=======================

Users can submit/manage analysis on PanDA using JupyterLab. This page explains how to setup JupyterLab
on your computer, how to install and setup panda-client on JupyterLab, and how to use client modules
and tools in Jupyter notebook.

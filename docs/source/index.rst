====================================
Welcome to PanDA's documentation
====================================

The Production and Distributed Analysis (PanDA) is a data-driven workload management system capable of
operating at massive data processing scale, designed to have
the flexibility to adapt to emerging computing technologies in processing, storage, networking and distributed
computing middleware.
